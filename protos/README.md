# protos/

Place `mailbox_server.proto` from the [`bark`](https://gitlab.com/ark-bitcoin/bark) repository (by [Second](https://second.tech)) here before starting the relay.

The relay reads this file at startup via `PROTO_PATH` (default: `./protos/mailbox_server.proto`).

This directory is intentionally kept out of version control. The proto file itself is owned by the [`bark`](https://gitlab.com/ark-bitcoin/bark) project by [Second](https://second.tech).
